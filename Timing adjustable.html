WCAG 2.2.1 – Timing Adjustable

This page demonstrates timed content that meets and fails WCAG requirements for adjustable time limits.

✅ Passing Example (Timing Adjustable)

Why this PASSES:
  • User is warned before time expires
  • User can extend the time
  • User can turn off the timer
Time remaining: 10 seconds
⚠ Time is about to expire!

❌ Failing Example (No Time Control)

Why this FAILS:
  • Times out automatically
  • No warning given
  • No ability to extend or disable time
Session will expire in 5 seconds
Accessibility Tip:
Time limits are acceptable only if users can turn them off, adjust them, or extend them. Exceptions apply only when timing is essential (e.g. auctions, real‑time games).
